Ruby环境的若干问题!!

Posted on 03 Aug 2013

本来不需要ruby环境的。为了安装cocoapods，结果引出一堆安装问题。焦头烂额了两天，现在终于搞定了。

不能download specs.4.8.gz

这个问题很坑爹，本来按照安装cocoapods的流程：

1     gem update --system
2     sudo gem install cocoapods

结果发现被墙，于是修改源： taobao镜像 但还是死活装不上，后来用

1     gem install cocoapods -V

才发现没有permission. 但是你要用sudo就说不能download specs.4.8，我了个擦，玩儿人不是。 决定不管sudo，于是修改文件夹权限，cocoapods顺利装上。

安装jekyll的问题

正常来说：

1     $ gem install jekyll

然后报错，得知ruby1.8.7版本太低，于是打算安装1.9.2

1     $ rvm install 1.9.2

报错说macports有问题（当时心中千万匹草泥马） 按照此贴操作无果（应该有用，只是没有解决我的问题）
